    注册

    用如下代码测试注册：

    # 通过shell，你可以直接使用curl来访问API
curl "http://qilian.jp/appointment/api/register"
  --form "mail=test@mobile.com&uuid=abcdefghijklmnopqrstuvwxyzabcdef"

    请确认测试邮箱为真实有效地邮箱，后续会有邮件发送确认. uuid为32为唯一随机串

    注册成功，返回结果:

    {
  "ok"
}

    注册失败，返回结果：

    {
"error"
}

    用户初始化APP时输入邮箱注册

    HTTP请求

    POST http://qilian.jp/appointment/api/register

    请求参数

    参数 默认值 备注
    mail 无 用户注册时使用的新邮箱
    uuid 无 手机端默认生成

    重置邮箱

    curl "http://qilian.jp/appointment/api/reset?uuid=abcdefghijklmnopqrstuvwxyzabcdef"

    重置成功，返回结果：

    您的邮箱已重置，请在手机端重新注册

    重置失败，返回结果：

    用户不存在，请在直接在手机端登录

    这个接口可以重置用户手机端的邮箱

    HTTP请求

    GET http://qilian.jp/appointment/api/reset

    请求参数

    参数 默认值 备注
    uuid 无 用户唯一标识码
